## 📘 Usage

### Install

```bash
make install # install all dependencies
make build # build UnCoin
```

### Run

```bash
make run
# or
./build/UnCoin-[version]
```

where `[version]` is the **UnCoin** current version

### Clean

```bash
make clean
```

this will remove all the compilation files
